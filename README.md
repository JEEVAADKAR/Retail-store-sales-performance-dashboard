## Overview
This project analyzes retail store sales data to track sales performance, profit trends, and product-wise contribution using Power BI.

## Tools Used
- Power BI
- Microsoft Excel
- GitHub

## KPIs
- Total Sales: 1.83M
- Total Profit: 96.75K
- Total Quantity Sold: 520

## Dashboard Insights
- Sales show monthly variation with peak performance in February and April
- Electronics category contributes the highest share of total sales
- Laptops and Tablets are top-selling products
- Certain customers generate significantly higher revenue
- Profit trends align with high-sales months

## Visuals Included
- Monthly Sales Trend
- Sales by Category
- Sales by Product
- Profit by Month
- Customer-wise Sales
- Interactive product slicer



## Conclusion
This dashboard helps stakeholders understand sales behavior and make data-driven decisions to improve profitability and inventory planning.
